EzCryptoAPI Test-Application Setup  
 
  1. Put the file rsaenh.dll into your system folder and register it using Regsvr32.exe [i.e.  command prompt: C:\windows\system\Regsvr32.exe rsaenh.dll ]. This DLL is the responsible of Triple DES, Triple DES 112, and DES [Be careful if you have intentions of creating applications using these algorithms 'cause you may be in trouble with the law!]
  2. Open Group1.vbg. This file should hold both, the test form and the control's full code
